# Weather Checker

Check weather information in any city, powered by Open Weather API.

### Stack:

- Nuxt.JS
- TailwindCSS
- JavaScript

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
